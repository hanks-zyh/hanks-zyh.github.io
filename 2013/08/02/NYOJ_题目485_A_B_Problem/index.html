NYOJ 题目485 A*B Problem

 

A*B Problem

时间限制：1000 ms  |  内存限制：65535 KB
难度：2
描述

设计一个程序求出A*B，然后将其结果每一位相加得到C，如果C的位数大于等于2，继续将C的各位数相加，直到结果是个一位数k。

例如：

6*8=48；

4+8=12；

1+2=3；

输出3即可。

输入
第一行输入一个数N(0<N<=1000000)，表示N组测试数据。
随后的N行每行给出两个非负整数m，n（0<=m,n<=10^12)。
输出
对于每一行数据，输出k。
样例输入
3
6 8
1234567 67
454 1232
样例输出
3
4
5

数学不好伤不起啊；；；

结果 =（A*B）%9   = （A%9 ）*(B%9) %9;

A,B 用long long 类型存放，，另外long long 类型要自己定义，否则结果会出错

 

 
#include <iostream>
#include <cstdio>
using namespace std;

typedef long long LL;//这里需要自己定义，直接定义不知道为什么会出错
LL a,b;
int main()
{
    int n;
 //   long long a,b;
    scanf("%d",&n);
    while(n--)
    {
        scanf("%lld%lld",&a,&b);
        if(a==0 || b==0)
            printf("0\n");
        else
        {
            int ans=(a%9)*(b%9)%9;
            if(ans==0) ans=9;
            printf("%d\n",ans);
        }
    }
    return 0;
}
