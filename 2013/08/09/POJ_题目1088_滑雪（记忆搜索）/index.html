POJ 题目1088 滑雪（记忆搜索）

 

滑雪
Time Limit: 1000MS Memory Limit: 65536K
Total Submissions: 67994 Accepted: 25014

Description

Michael喜欢滑雪百这并不奇怪， 因为滑雪的确很刺激。可是为了获得速度，滑的区域必须向下倾斜，而且当你滑到坡底，你不得不再次走上坡或者等待升降机来载你。Michael想知道载一个区域中最长底滑坡。区域由一个二维数组给出。数组的每个数字代表点的高度。下面是一个例子 
 1  2  3  4 5

16 17 18 19 6

15 24 25 20 7

14 23 22 21 8

13 12 11 10 9

一个人可以从某个点滑向上下左右相邻四个点之一，当且仅当高度减小。在上面的例子中，一条可滑行的滑坡为24-17-16-1。当然25-24-23-...-3-2-1更长。事实上，这是最长的一条。

Input

输入的第一行表示区域的行数R和列数C(1 <= R,C <= 100)。下面是R行，每行有C个整数，代表高度h，0<=h<=10000。

Output

输出最长区域的长度。

Sample Input

5 5
1 2 3 4 5
16 17 18 19 6
15 24 25 20 7
14 23 22 21 8
13 12 11 10 9

 

1.从每个点都要搜索，将该点四个方向搜素的最大长度 记录在vis[][],相应的位置，如：（1,1）点（17那个） 四个方向最大长度为 3，则在vis[1][1]中存3

2,.注意每次都要几率最大值max，并且，每次更新

3.超出边界的情况

 

#include<stdio.h>
#include<string.h>

int R,C;
int a[101][101],vis[101][101];

int DFS(int i,int j)
{
	int max=0,tem=0;
	if(i<0 || i+1 >R || j<0 || j+1> C) return 0;
	
	if(vis[i][j]) return vis[i][j];
	
	if( a[i+1][j] <a[i][j] )
	{
		tem = DFS(i+1,j);
		max = max>tem?max:tem;
	}
	if( a[i-1][j] <a[i][j] )
	{
		tem = DFS(i-1,j);
		max = max>tem?max:tem;
	}
	if( a[i][j-1] <a[i][j] )
	{
		tem = DFS(i,j-1);
		max = max>tem?max:tem;
	}
	if( a[i][j+1] <a[i][j] )
	{
		tem = DFS(i,j+1);
		max = max>tem?max:tem;
	}
	
	return  vis[i][j] = max+1;
	
}
int main()
{
	int i,j,max,tem;
	while(scanf("%d%d",&R,&C)!=EOF)
	{
		memset(vis,0,sizeof(vis));
		memset(a,0,sizeof(a));
		for(i=0;i<R;i++)
			for(j=0;j<C;j++)
				scanf("%d",&a[i][j]);
		max = 0;
		for(i=0;i<R;i++)
			for(j=0;j<C;j++)
			{
				tem = DFS(i,j);
				max = max>tem?max:tem;	
			}
		printf("%d\n",max);
	} 
	return 0;
}
