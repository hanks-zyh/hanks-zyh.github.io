hdu 题目1596 find the safest road （最短路径 dijkstra）

find the safest road

Time Limit: 10000/5000 MS (Java/Others)    Memory Limit: 32768/32768 K (Java/Others)
Total Submission(s): 4943    Accepted Submission(s): 1797


Problem Description

 

XX星球有很多城市，每个城市之间有一条或多条飞行通道，但是并不是所有的路都是很安全的，每一条路有一个安全系数s,s是在 0 和 1 间的实数(包括0，1)，一条从u 到 v 的通道P 的安全度为Safe(P) = s(e1)*s(e2)…*s(ek) e1,e2,ek是P 上的边 ，现在8600 想出去旅游，面对这这么多的路，他想找一条最安全的路。但是8600 的数学不好，想请你帮忙 ^_^
 


 

Input

 

输入包括多个测试实例，每个实例包括：
第一行：n。n表示城市的个数n<=1000;
接着是一个n*n的矩阵表示两个城市之间的安全系数，(0可以理解为那两个城市之间没有直接的通道)
接着是Q个8600要旅游的路线,每行有两个数字，表示8600所在的城市和要去的城市
 


 

Output

 

如果86无法达到他的目的地，输出"What a pity!",
其他的输出这两个城市之间的最安全道路的安全系数,保留三位小数。
 


 

Sample Input

 

3
1 0.5 0.5
0.5 1 0.4
0.5 0.4 1
3
1 2
2 3
1 3
 


 

Sample Output

 

0.500
0.400
0.500
 



 

 

 最短路径问题。不过这道题是求最大值的，原理一样、、、

#define N 1005
#include<stdio.h>

double G[N][N],dis[N];
bool vis[N];
int n,t;

void dijkstra(int s){
	
	int i,j;	
	for(i=1;i<=n;i++){//初始化
		vis[i]=0;
		dis[i] = G[s][i]*G[s][s];
	}
	vis[s] = 1;	dis[s] = G[s][s];
	for(i=1;i<=n;i++){
		
		double max =0; //double 一开始忘了，花了大量时间找错误了 
		int last ;
		for(j=1;j<=n;j++){//寻找当前最大安全系数
			if(!vis[j] && dis[j]>max){
				max = dis[j];
				last = j; 
			} 
		}
		vis[last] = 1;
		for(j=1;j<=n;j++){	//跟新dis【】		
			if(!vis[j]){		
				double newdis = dis[last]*G[last][j];	//还有这里 d ouble	
				if(dis[j] < newdis )	dis[j] = newdis;												
			}		
		}	
	}
	if(dis[t]<0.00000000001) printf("What a pity!\n");
	else printf("%.3lf\n",dis[t]);
	
	
}

int main()
{
	int s,d,i,j,q;	
	while(scanf("%d",&n)!=EOF)
	{
		for(i=1;i<=n;i++){
			for(j=1;j<=n;j++)
				scanf("%lf",&G[i][j]);
		}	
		scanf("%d",&q);
		while(q--){
			scanf("%d%d",&s,&t);
			dijkstra(s); 
		}		
	}
	return 0;
}
