初学者的烦恼

时间限制：1000 ms  |  内存限制：65535 KB
难度：1
描述

输入
第一行包括一个整数t，代表有t组数据；
每组数据第一行包括一个整数m(1<=m<=50),第二行有m个小写英文字符，第三行包括一个整数n（0<=n<=100），代表字符串向左移动n位。
输出
每组数据输入站一行，即输出移动后的字符串。

5
asdfk
0
