Mac 上利用 launchctl 开启定时任务

前几天写了 Python 脚本进行爬虫，现在利用 launchctl 来定时开启爬虫任务，launchctl 通过 plist 属性列表（Property List）配置。Linux 和 Mac 通用的 crontab 时间间隔可以精确到分钟，而launchctl 时间间隔可以精确到每秒。
