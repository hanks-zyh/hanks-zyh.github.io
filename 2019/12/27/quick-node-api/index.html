利用 Node 快速创建并上线接口

演示快速创建并上线一个 API 服务。

准备

  1. 服务器（腾讯云、阿里云等）
  2. Node 环境，VS Code （安装 SSH 插件并配置连接到服务器）

开发

接口文件，app.js

监听端口为 9877，收到请求会返回一个 hello。
运行 node app.js，服务开起来，可用通过 http://localhost:9877 进行访问了

上线

需要将接口在服务器运行起来，利用 pm2 来运行。

外网访问

一般的服务器会有安全策略，只允许指定的端口进入。所以还需在服务器的配置上打开执行的端口即可。
